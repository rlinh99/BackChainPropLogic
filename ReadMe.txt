Running Instruction:
Please put the test data with main.py in the same folder, and run main.py
Please just enter the file name, without quotation mark.
Please enter a single item as query. 
The program automatically reruns, so you can try other queries after current query is done. 
Exit program by entering any non-alphabetical character or just hitting "Enter".
